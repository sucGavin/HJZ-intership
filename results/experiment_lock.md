## 模型公式、约束与成本计算方式

调度时段为 \(t=1,2,\ldots,T\)，时间间隔为 \(\Delta t\)。\(P_{w,t}\)、\(P_{pv,t}\)、\(P_{hr,t}\)、\(P_{L,t}\) 分别表示风电最大可发功率、光伏最大可发功率、余热发电功率和负荷功率，\(\pi_t\) 为分时电价。储能功率 \(P_{b,t}>0\) 表示充电，\(P_{b,t}<0\) 表示放电，\(E_t\) 为储能电量，\(P_{g,t}\) 为电网购电功率。

购电功率计算：

```text
P_g,t = max(0, P_L,t + max(P_b,t,0) - max(-P_b,t,0) - P_w,t - P_pv,t - P_hr,t)
```

储能状态更新：

```text
E_t+1 = E_t + eta_ch * max(P_b,t,0) * Delta t - max(-P_b,t,0) * Delta t / eta_dis
eta_ch = eta_dis = sqrt(0.8)
```

约束条件：

```text
-30 <= P_b,t <= 30
0 <= E_t <= 60
P_g,t >= 0
E_1 = 30 MWh
E_T >= E_1
```

成本计算方式：

```text
C_total = C_grid + C_bat
C_grid = 0.1 * sum(pi_t * P_g,t * Delta t)
C_bat = 0.1 * c_bat * sum(|P_b,t| * Delta t)
```

其中综合运行成本由购电成本和储能等效运行成本组成；储能等效运行成本仅用于复算综合运行成本，不进入核心结果表。
