## 摘要可用结论

- 主实验为典型日 96 点调度对比，背景数据为 7 天 672 点仿真序列。
- 对比策略包括逻辑判断基准策略、PSO 和 DE。
- 在锁定结果中，PSO 的综合运行成本为 76.6853 万元，低于逻辑判断基准策略的 77.3168 万元和 DE 的 77.7295 万元。

## 第 2 章可写素材：简化调度模型、公式和约束

变量含义：

- \(P_{w,t}\)：风电最大可发功率。
- \(P_{pv,t}\)：光伏最大可发功率。
- \(P_{hr,t}\)：余热发电功率。
- \(P_{L,t}\)：计划负荷功率。
- \(P_{b,t}\)：储能功率，正值表示充电，负值表示放电。
- \(E_t\)：储能电量。
- \(P_{g,t}\)：电网购电功率。
- \(\pi_t\)：分时电价。

购电功率公式：

```text
P_g,t = max(0, P_L,t + max(P_b,t,0) - max(-P_b,t,0) - P_w,t - P_pv,t - P_hr,t)
```

储能状态更新公式：

```text
E_t+1 = E_t + eta_ch * max(P_b,t,0) * Delta t - max(-P_b,t,0) * Delta t / eta_dis
```

约束条件：

```text
P_b,min <= P_b,t <= P_b,max
E_min <= E_t <= E_max
P_g,t >= 0
E_1 = E_initial
E_T >= E_1
```

成本计算方式：

```text
C_total = C_grid + C_bat
C_grid = unit_conversion_factor * sum(pi_t * P_g,t * Delta t)
C_bat = unit_conversion_factor * c_bat * sum(|P_b,t| * Delta t)
```

写作注意事项：

- 核心结果表中只展示风电最大可发电量、光伏最大可发电量、余热发电量、计划负荷用电量、购电成本、综合运行成本和算法运行时间。
- 储能等效运行成本仅用于校验报告中的综合运行成本复算说明，不作为核心结果表指标。

- 结论 1：在典型日 96 点调度对比中，PSO 的综合运行成本为 76.6853 万元，低于逻辑判断基准策略的 77.3168 万元和 DE 的 77.7295 万元；5 次运行统计中，PSO 的平均综合运行成本为 76.8220 万元，DE 为 77.8939 万元。
