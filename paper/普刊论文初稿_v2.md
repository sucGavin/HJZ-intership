# 基于改进粒子群与差分进化算法的源网荷储一体化园区多时间尺度经济调度仿真研究

## 摘要

针对源网荷储一体化园区中风电、光伏、余热发电、储能和电网购电之间的协调调度问题，本文构建简化多时间尺度经济调度仿真模型，并以典型日 96 点调度对比作为主实验口径。背景数据采用 7 天 672 点仿真序列，算法对比策略包括逻辑判断基准策略、粒子群算法（PSO）和差分进化算法（DE）。模型以购电成本和储能等效运行成本构成综合运行成本，通过储能充放电约束、购电功率约束和末端电量约束保证调度结果满足基本运行条件。仿真结果表明，在当前典型日仿真算例下，PSO 的综合运行成本为 76.6853 万元，低于逻辑判断基准策略的 77.3168 万元和 DE 的 77.7295 万元；5 次运行统计中，PSO 的平均运行时间为 1.6974 s，低于 DE 的 6.9302 s，PSO 的标准差为 0.1097，低于 DE 的 0.1576。结果说明，在本文锁定仿真条件下，PSO 在经济性、计算效率和稳定性方面表现更优。

**关键词**：源网荷储；多时间尺度调度；粒子群算法；差分进化算法；经济调度

## 1 源网荷储一体化园区调度要求及策略整体框架

源网荷储一体化园区调度需要同时考虑新能源出力波动、园区计划负荷需求、余热发电连续出力、储能充放电调节能力以及电网购电成本。风电和光伏具有波动性，余热发电具有相对连续的出力特征，计划负荷则体现园区用能需求。储能系统可在低电价或新能源出力较高时段充电，在高电价或负荷压力较高时段放电，从而改善购电功率分布并降低综合运行成本。

本文采用多时间尺度调度框架展开仿真分析。日前层面基于背景仿真序列形成基础调度信息，日内层面围绕典型日 96 点数据进行优化对比，实时层面体现短时偏差下的逻辑修正思想。需要说明的是，本文主实验口径固定为典型日 96 点调度对比，7 天 672 点仿真序列仅作为背景数据，不作为 PSO 与 DE 的主对比口径。

对比策略包括逻辑判断基准策略、PSO 和 DE。逻辑判断基准策略用于提供可解释的基准调度结果；PSO 和 DE 用于搜索储能充放电序列，使综合运行成本降低。三种策略使用同一组典型日数据，因此风电最大可发电量、光伏最大可发电量、余热发电量和计划负荷用电量在核心结果表中保持一致。

## 2 源网荷储一体化园区多时间尺度调度模型

设调度时段为 \(t=1,2,\ldots,T\)，时间间隔为 \(\Delta t\)。\(P_{w,t}\)、\(P_{pv,t}\)、\(P_{hr,t}\)、\(P_{L,t}\) 分别表示风电最大可发功率、光伏最大可发功率、余热发电功率和负荷功率，\(\pi_t\) 为分时电价。储能功率 \(P_{b,t}>0\) 表示充电，\(P_{b,t}<0\) 表示放电，\(E_t\) 为储能电量，\(P_{g,t}\) 为电网购电功率。

购电功率计算如下：

```text
P_g,t = max(0, P_L,t + max(P_b,t,0) - max(-P_b,t,0) - P_w,t - P_pv,t - P_hr,t)
```

该式表示风电、光伏和余热发电优先满足负荷与储能充电需求。当可用出力不足时，由电网购电补足；当可用出力超过负荷和充电需求时，购电功率为 0。

储能状态更新如下：

```text
E_t+1 = E_t + eta_ch * max(P_b,t,0) * Delta t - max(-P_b,t,0) * Delta t / eta_dis
eta_ch = eta_dis = sqrt(0.8)
```

其中 \(eta_{ch}\) 和 \(eta_{dis}\) 分别表示充电效率和放电效率。储能在充电时电量增加，在放电时电量减少，状态更新同时考虑效率损耗。

约束条件如下：

```text
-30 <= P_b,t <= 30
0 <= E_t <= 60
P_g,t >= 0
E_1 = 30 MWh
E_T >= E_1
```

其中，储能功率约束限制充放电功率范围，电量约束限制储能状态范围，购电功率约束保证电网购电非负，末端电量约束用于避免调度周期末过度放电导致成本低估。

成本函数如下：

```text
C_total = C_grid + C_bat
C_grid = 0.1 * sum(pi_t * P_g,t * Delta t)
C_bat = 0.1 * c_bat * sum(|P_b,t| * Delta t)
```

其中 \(C_{total}\) 为综合运行成本，\(C_{grid}\) 为购电成本，\(C_{bat}\) 为储能等效运行成本。核心结果表只展示购电成本和综合运行成本，储能等效运行成本仅用于复算综合运行成本。

### 2.1 日前调度策略

日前调度用于形成园区在较长时间尺度上的基础运行安排。本文中，7 天 672 点仿真序列作为背景数据，为风电、光伏、余热发电和计划负荷的变化特征提供仿真基础。日前层面不作为本文 PSO 与 DE 的主对比口径，而是为典型日调度分析提供背景。

在日前策略中，可根据分时电价和源荷变化趋势形成储能充放电的初步方向：低电价或新能源出力较高时段优先考虑充电，高电价或负荷较高时段优先考虑放电。该策略为后续日内优化提供基础逻辑。

### 2.2 日内调度策略

日内调度是本文算法对比的核心层面。主实验固定为典型日 96 点调度对比，三种策略均在同一组典型日数据上运行。PSO 和 DE 以储能充放电序列为搜索对象，通过成本函数评价候选调度方案，并在功率、电量和末端电量约束下获得对应调度结果。

日内调度的重点在于根据分时电价和源荷出力关系调整储能功率。当低电价时段或源侧出力较高时，储能充电可为后续高电价时段提供可用电量；当高电价时段或负荷需求较高时，储能放电可减少电网购电量，从而降低综合运行成本。

### 2.3 实时调度策略

实时调度主要体现短时间尺度下对功率偏差的修正思想。由于本文为仿真研究，实时层面不引入新的优化目标或新增评价指标，而是在已有模型框架下说明调度逻辑：当源侧出力或负荷需求出现短时偏差时，优先利用储能在功率和电量约束内进行修正，无法由储能完全补偿的部分由电网购电补足。

实时调度与日前、日内调度共同构成多时间尺度协调框架。日前层面提供背景计划，日内层面完成典型日优化对比，实时层面强调储能对短时偏差的调节作用。

## 3 算例分析

### 3.1 参数设置

本文算例采用锁定仿真条件开展分析。调度模型包含风电、光伏、余热发电、计划负荷、储能和电网购电。储能功率约束、电量约束、初始电量约束和末端电量约束按照实验锁定文件中的模型设置执行。成本函数由购电成本和储能等效运行成本构成，其中综合运行成本作为不同策略比较的主要成本结果。

算法对比策略为逻辑判断基准策略、PSO 和 DE。PSO 与 DE 均用于优化典型日储能充放电序列，三种策略使用同一组典型日 96 点数据。所有实验数值均引用锁定结果，不自行修改或推算。

### 3.2 仿真数据构建

本文背景数据为 7 天 672 点仿真序列，主实验为典型日 96 点调度对比。典型日数据用于计算风电最大可发电量、光伏最大可发电量、余热发电量和计划负荷用电量。上述四项能量统计均基于同一组典型日数据，因此在逻辑判断基准策略、PSO 和 DE 三行结果中完全一致。

本文不使用真实工程数据，算例分析仅围绕锁定仿真数据和锁定结果展开。7 天 672 点数据仅作为仿真背景数据保留，不用于改变主实验对比口径。

### 3.4 多时间尺度调度效果分析

多时间尺度调度的作用体现在储能充放电与分时电价响应的协调。日前层面根据背景仿真序列提供基础计划，日内层面在典型日 96 点数据上优化储能充放电，实时层面通过储能对短时偏差进行逻辑修正。三者共同服务于降低购电成本和综合运行成本的调度目标。

从锁定结果看，PSO 通过调整储能充放电序列，使综合运行成本低于逻辑判断基准策略和 DE。该结果说明，在当前仿真算例中，储能可通过低价或源侧出力较高时段充电、高价或负荷较高时段放电的方式改善购电功率分布。需要强调的是，本文不使用真实工程数据，结果仅对应当前锁定仿真算例。

## 4 结论

1. 当前典型日仿真算例下，PSO 在经济性、计算效率和稳定性方面表现更优。PSO 综合运行成本为 76.6853 万元，低于逻辑判断基准策略的 77.3168 万元和 DE 的 77.7295 万元；PSO 平均运行时间为 1.6974 s，低于 DE 的 6.9302 s；PSO 标准差为 0.1097，低于 DE 的 0.1576。

2. 多时间尺度调度通过储能充放电与分时电价响应，可以改善园区经济运行效果。日前背景计划、日内典型日优化和实时逻辑修正共同发挥作用，使储能能够在满足约束条件的基础上参与购电成本优化。
